# Student Management System

## Overview

The Student Management System is a desktop-based application developed using Java Swing, JDBC, and MySQL. It helps educational institutions manage student records, attendance, marks, and report cards through an easy-to-use graphical interface.

## Features

### Admin Module

* Admin Login
* Add Student
* View Students
* Update Student Details
* Delete Student Records
* Search Student
* Attendance Management
* Attendance Analytics
* Marks Entry
* View Marks
* Report Card Generation

### Staff Module

* Staff Login
* View Students
* Attendance Management
* Report Card Access

## Technologies Used

* Java
* Java Swing

* MySQL
